## Ticket: Source resolver and image output utilities

Add two utility functions to `src/aprilcam/mcp_server.py`.

### 1. `resolve_source(source_id) -> np.ndarray`

Add this as a regular function (not async, not a tool) in mcp_server.py. It should:
- First try `playfield_registry.get(source_id)` — if found, get the underlying camera via `registry.get(pf_entry.camera_id)`, read a frame, apply `pf_entry.playfield.deskew(frame)`, return the deskewed frame
- Then try `registry.get(source_id)` — if found, read a frame, return it
- If neither found, raise `KeyError(f"Unknown source_id '{source_id}'")`
- If frame read fails, raise `RuntimeError("Failed to read frame")`

```python
def resolve_source(source_id: str) -> np.ndarray:
    """Resolve a source_id to a BGR frame. Checks playfield first, then camera."""
    import cv2
    # Try playfield
    try:
        pf_entry = playfield_registry.get(source_id)
        cap = registry.get(pf_entry.camera_id)
        ret, frame = cap.read()
        if not ret:
            raise RuntimeError("Failed to read frame from camera")
        return pf_entry.playfield.deskew(frame)
    except KeyError:
        pass
    # Try camera
    try:
        cap = registry.get(source_id)
        ret, frame = cap.read()
        if not ret:
            raise RuntimeError("Failed to read frame from camera")
        return frame
    except KeyError:
        raise KeyError(f"Unknown source_id '{source_id}'")
```

### 2. `format_image_output(frame, format="base64", quality=85) -> list[TextContent | ImageContent]`

```python
def format_image_output(
    frame: np.ndarray,
    format: str = "base64",
    quality: int = 85,
) -> list[TextContent | ImageContent]:
    """Encode a BGR frame as base64 ImageContent or write to a temp file."""
    import cv2
    ok, buf = cv2.imencode(".jpg", frame, [cv2.IMWRITE_JPEG_QUALITY, quality])
    if not ok:
        return [TextContent(type="text", text=json.dumps({"error": "Failed to encode image"}))]
    if format == "file":
        tmp = tempfile.NamedTemporaryFile(suffix=".jpg", delete=False)
        tmp.write(buf.tobytes())
        tmp.close()
        return [TextContent(type="text", text=json.dumps({"path": tmp.name}))]
    b64 = base64.b64encode(buf.tobytes()).decode("ascii")
    return [ImageContent(type="image", data=b64, mimeType="image/jpeg")]
```

Place both functions after the registry definitions but before the tool functions.
