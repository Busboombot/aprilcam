## Ticket: Add aprilcam init subcommand to configure MCP files

### What it does

`aprilcam init` looks in the current working directory and adds/updates the aprilcam MCP server entry in two config files:

1. **`.mcp.json`** (Claude Code format):
```json
{
  "mcpServers": {
    "aprilcam": {
      "command": "aprilcam",
      "args": ["mcp"]
    }
  }
}
```

2. **`.vscode/mcp.json`** (VS Code format):
```json
{
  "servers": {
    "aprilcam": {
      "type": "stdio",
      "command": "aprilcam",
      "args": ["mcp"]
    }
  }
}
```

### Critical: Must not clobber existing entries

These files may already have other MCP servers configured. The init command must:
1. Read the existing file if it exists (or start with empty structure if not)
2. Parse as JSON
3. Insert/update only the `"aprilcam"` key under the appropriate parent key
4. Write back the full JSON with `indent=2`
5. Create `.vscode/` directory if needed

If the file doesn't exist, create it with just the aprilcam entry.
If the file exists but is empty or invalid JSON, warn and create fresh.

### Implementation

1. **Create `src/aprilcam/cli/init_cli.py`**:

```python
import json
from pathlib import Path

CLAUDE_CODE_ENTRY = {
    "command": "aprilcam",
    "args": ["mcp"]
}

VSCODE_ENTRY = {
    "type": "stdio",
    "command": "aprilcam",
    "args": ["mcp"]
}

def _upsert_mcp_config(path: Path, parent_key: str, entry: dict) -> None:
    """Read JSON file, insert/update aprilcam entry, write back."""
    data = {}
    if path.exists():
        try:
            text = path.read_text().strip()
            if text:
                data = json.loads(text)
        except json.JSONDecodeError:
            print(f"  Warning: {path} contains invalid JSON, overwriting")
            data = {}
    
    if parent_key not in data:
        data[parent_key] = {}
    
    data[parent_key]["aprilcam"] = entry
    
    path.parent.mkdir(parents=True, exist_ok=True)
    path.write_text(json.dumps(data, indent=2) + "\n")

def main(argv=None):
    cwd = Path.cwd()
    
    # .mcp.json (Claude Code)
    mcp_path = cwd / ".mcp.json"
    _upsert_mcp_config(mcp_path, "mcpServers", CLAUDE_CODE_ENTRY)
    print(f"  Updated {mcp_path}")
    
    # .vscode/mcp.json (VS Code)
    vscode_path = cwd / ".vscode" / "mcp.json"
    _upsert_mcp_config(vscode_path, "servers", VSCODE_ENTRY)
    print(f"  Updated {vscode_path}")
    
    print("AprilCam MCP server configured. Restart your editor to connect.")
    return 0
```
